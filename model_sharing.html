BioModels

Models can be shared with other users by their owners (the users who submitted the model) and by users with administrator rights. Users with sharing rights can click the share model icon on the display page, leading to the screen shown below.

By typing in the first few characters of the name of the user that is to be granted access to the model an autocomplete query is triggered, and the repository offers suggestions. When the add button is clicked the user is validated on the server, and if a user exists with that name they are added to the list.

The list of users on the right specifies the permissions granted to them. These can be modified, or the user's access can be revoked entirely by clicking the remove button.

Please don't forget to save the changes made to the model by clicking 'Save'.
